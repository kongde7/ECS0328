# 我怎么判断结果有没有遗漏？
4224台vm释放时间为“\N”，只有running一行。  
128台vm在3月29日后释放，而表格3月28日截止，也只有running一行。  
195648台vm正常创建和释放，running和release各输出一行。  

4224 +128 + 195648 = 200000 行，刚好符合输入样本。  
4224 +128 + 195648 * 2 = 385648 行，刚好符合输出的vm.csv文件。
